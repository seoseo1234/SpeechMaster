# 🎯 스피치마스터 프로(SpeechMaster Pro) 최종 산출물 제출

## 📘 개요
**스피치마스터 프로** 
본 프로젝트는 초등학생(저학년/고학년) 대상 맞춤형 AI 기반 스피치 진단 및 코칭 웹앱으로, 음성(성량/발음), 시각(시선/태도), 언어(속도/습관어) 데이터를 다차원 분석하여 **초등학생의 낭독 및 발표 능력을 향상시키는 멀티모달(Multimodal) 스피치 훈련 포털**입니다.

## 📖 사용 방법
1. 교사는 구글 계정으로 로그인하여 학급 대시보드를 생성하고 확인합니다.
2. 학생은 접속 후, 자신의 학년에 맞는 모드(저학년/고학년)를 선택합니다.
3. **[저학년 모드]** 화면에 제시된 지문(동화책/문장)을 마이크에 대고 읽으면, AI가 발음의 정확도를 실시간으로 분석하여 피드백합니다.
4. **[고학년 모드]** 웹캠과 마이크를 켜고 발표를 진행하면, 시선 이탈 여부와 성량 크기, 습관어 사용 등을 실시간으로 측정하여 코칭해 줍니다.
5. 연습이 끝나면 발음 점수, 성량, 시선 안정도 등이 포함된 직관적인 결과 리포트와 응원 메시지를 확인합니다.
6. 교사는 전용 관리 대시보드에서 학생별 누적 성장 데이터를 확인하고, AI가 작성해 준 학생별 서술형 관찰평가 문구를 나이스(NEIS) 입력에 바로 활용할 수 있습니다.

## 🌟 프로젝트 의의
이 웹앱은 초등학교 현장에서 겪고 있는 저학년의 '한글 해득 부진'과 고학년의 '발표 불안' 문제를 첨단 AI(멀티모달) 기술을 통해 효과적으로 해결하고자 기획되었습니다. 기존에는 교사가 일일이 파악하고 기록하기 어려웠던 학생들의 미세한 발음 오류, 성량 변화, 시선 처리 등을 객관적인 데이터로 진단함으로써 더욱 정교한 개별화 맞춤 지도가 가능해집니다.

또한, 학생들은 게이미피케이션 요소가 가미된 직관적인 피드백을 통해 흥미를 잃지 않고 스스로 낭독과 발표 연습을 주도할 수 있습니다. 교사에게는 AI 어시스턴트가 객관적 데이터 기반의 서술형 평가 문구를 추천해 줌으로써 교육 행정 업무 부담을 크게 경감시키고, 본연의 교육에 집중할 수 있도록 돕습니다. 향후에는 다양한 교과서 대본 및 맞춤형 취약점 보완 훈련 코스를 추가하여 공교육 현장에서의 활용도를 더욱 높일 예정입니다.
